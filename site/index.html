Hen Harrier Tracker

Automated tracking of UK satellite-tagged hen harrier status, cross-referenced against mapped moorland/upland habitat. Every marker traces back to a dated, linked source. About the source data

Locations not disclosed by NE

Natural England deliberately withholds precise roost/nest sites for some birds for species-protection reasons. This is not a data gap in this pipeline.

      Data freshness

        This is a batch pipeline on a schedule, not a real-time feed — see each source's own refresh cadence above. Habitat cross-referencing shows proximity to mapped moorland/upland habitat as a proxy, not proof of a grouse moor and never a named estate or owner. Nothing is published as a confirmed habitat match until a human reviewer has checked the evidence. Inspired by (not a replication of) Murgatroyd et al. 2019.
